## პროექტი **"ინფო კალენდარი"**
---
#### ეს არის საგანმანათლებლო ჟურნალი, სადაც გთავაზობთ ყოველდღიურ ინფორმაციას ათასობით მსოფლიო დღესასწაულების შესახებ, რომლებიც კონკრეტულ დღეს უკავშირდება:

1. დღესასწაულები:
- საქართველოს დღესასწაულები.
- მსოფლიო დღესასწაულები.
- ქვეყნების და ქალაქების დღეები.
- მართლმადიდებლური ეკლესიის წმინდანების ხსენების დღეები.

2. მნიშვნელოვანი ისტორიული მოვლენები:
- ქვეყნების დამოუკიდებლობის, გაერთიანების და დაშლის დღეები
- იმპერატორები, მეფეები, პრეზიდენტები
- გამოგონებები და აღმოჩენები
- ომები

3. ცნობილი ადამიანებისა და ისტორიული პიროვნებების დაბადების თარიღები.

4. ცნობილი ადამიანებისა და ისტორიული პიროვნებების გარდაცვალების თარიღები.
---
#### ინფორმაციის მარტივად აღსაქმელად სტატიები დაყოფილია შესაბამის კატეგორიებად: 
1. დღესასწაულები
2. ისტორიული ფაქტები
3. დღეს დაბადებულნი
4. დღეს გარდაცვლილნი

ინფორმაცია აღებულია ინგლისურენოვანი და ქართულენოვანი ვიკიპედიიდან
