ათასობით სარედაქციო სტატია დღესასწაულების, მნიშვნელოვანი მოვლენებისა და თარიღების შესახებ ყოველდღიურად.
პროექტი "მოვლენების კალენდარი" არის ყოველდღიური საგანმანათლებლო ჟურნალი, რომელიც გთავაზობთ ინფორმაციას მსოფლიო დღესასწაულების, ქალაქის დღეების, სახელობის დღეების, ცნობილი ადამიანებისა და ისტორიული პერსონაჟების ცხოვრებაში მნიშვნელოვანი თარიღებისა და სხვა მნიშვნელოვანი მოვლენების შესახებ, რომლებიც დღევანდელ დღეს უკავშირდება.
